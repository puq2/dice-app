# dice-app

The Dice Rolling App is a simple but versatile app that can be used to play many different games with people. Many people may want to use a set of dice for things such as board games but can easily lose the original dice that came with the games. The simple dice app allows you to pull out your phone, and with a tap of a button roll some dice.
